# 상세 구현 갭 분석

## ❌ 실제 미구현/미흡 사항 상세 분석

### 1. 채널 헤더 모달들 - 대부분 기본 구조만 존재

#### ❌ **채널 내 검색** (`SearchModalContent.jsx`)
**현재**: 검색창만 있음 (입력 필드 1개)
**필요**: 
- 검색 결과 표시
- 검색 히스토리
- 검색 필터 (날짜, 작성자 등)
- 실제 검색 기능 연동

#### ❌ **채널 참여자 목록** (`MembersModalContent.jsx`)  
**현재**: 전체 users를 단순 리스트로만 표시
**필요**:
- ❌ 게스트 초대 버튼
- ❌ 채널 참여자 검색
- ❌ 참여자 유형별 분류 (멤버/게스트/관리자)
- ❌ 실제 채널별 참여자만 필터링 (현재 전체 users 표시)

#### ⚠️ **고정된 메시지** (`PinnedModalContent.jsx`)
**현재**: messages에서 pinned 필터링만 함
**필요**:
- ✅ 기본 기능은 있음
- ❌ 고정 해제 기능
- ❌ 고정 날짜/사용자 정보

#### ⚠️ **스레드 목록** (`ThreadsModalContent.jsx`)
**현재**: threadId 있는 메시지만 표시
**필요**:
- ✅ 기본 목록 표시
- ❌ 스레드 답글 개수 표시
- ❌ 마지막 답글 시간
- ❌ 읽지 않은 답글 표시

#### ⚠️ **채널 파일 목록** (`ChannelFilesModalContent.jsx`)
**현재**: 탭만 있고 "비어있음" 메시지만 표시
**필요**:
- ❌ 실제 파일 리스트
- ❌ 파일 다운로드
- ❌ 파일 미리보기
- ❌ 파일 검색

#### ❌ **채널 정보** (`InfoModalContent.jsx`)
**현재**: 제목과 설명만 표시
**필요**:
- 채널 이름
- 채널 설명/토픽
- 채널 생성일
- 채널 생성자
- 참여자 수
- ❌ 채널 설정 편집 기능 (관리자만)
- ❌ 채널 나가기 버튼
- ❌ 채널 알림 설정

**현재**: MessageForwardModal.jsx 파일은 있음
**확인 필요**: 실제 연동 상태

### 5. 워크스페이스 설정 페이지

#### 실제 확인 필요 (WorkspaceSettingsPage.jsx 700+ 라인)
각 탭이 실제로 얼마나 구현되었는지 확인 필요:

1. **멤버 관리 탭** - 확인 필요
   - 참여자 역할 변경 기능 구현 여부
   - 차단 기능 구현 여부
   - 승인 대기자 처리 기능

2. **초대 관리 탭** - 확인 필요
   - 초대 취소 기능
   - 링크 강제 만료 기능

3. **그룹 관리 탭** - 확인 필요
   - 그룹별 채널 접근 권한
   - 그룹별 멤버 관리

4. **연동 관리 탭** - 시나리오에도 "보완 필요"
   - 외부 도구 목록
   - 연동 설정
   - 연동 해제

5. **보안 설정 탭** - 시나리오에도 "보완 필요"
   - 2FA 설정
   - IP 제한
   - 세션 타임아웃

6. **감사 로그 탭** - 시나리오에도 "보완 필요"
   - 로그 필터링
   - 로그 검색
   - CSV 내보내기
