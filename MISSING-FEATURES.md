### 채널 페이지

#### ❌ 채널 헤더 - 시나리오 요구사항 vs 현재
**시나리오**:
```
1) 채널 헤더
[1] 해당 채널 푸시 알림 설정
[2] 채널 내 검색 (카톡 채팅방 검색 기능 참고)
[3] 채널 참여자 목록
  (1) 게스트 초대 버튼
  (2) 채널 참여자 검색
  (3) 참여자 유형별 분류 선택
  (4) 채널 참여자 조회 목록
[4] 고정된 메세지
[5] 스레드 목록
[6] 채널 내 첨부된 파일 목록
[7] 채널 정보
```

**실제 필요한 기능**:
- [3] 채널 참여자 목록 안에 (1)(2)(3)(4)가 다 들어가야 함
- MembersModalContent.jsx는 수정했지만 GenericModal.jsx에서 연결 확인 필요

### 워크스페이스 설정 페이지

시나리오 상세:
```
** 워크스페이스 설정 페이지 : workspace owner, workspace manager **
[좌측 사이드바 레이아웃]
1. 워크스페이스 인사이트 탭
2. 멤버 관리 탭
3. 초대 관리 탭
4. 그룹 관리 탭
5. 연동 관리 탭
6. 보안 설정 탭
7. 감사 로그 탭
8. AI 관리 탭

[메인 레이아웃]
1. 워크스페이스 인사이트 탭 (기존의 개요 대시보드)
  [1] 워크스페이스 정보
  [2] 주요 모니터링 카드

2. 멤버 관리 탭
  [1] 참여자 (역할 유형 변경 가능)
  [2] 차단된 사용자
  [3] 입장/탈퇴 내역
  [4] 매니저 승인 대기자

3. 초대 관리 탭
  [1] 대기중인 초대
  [2] 재사용 가능한 초대 링크 (여기서 강제 만료시키는 것도 가능)

4. 그룹 관리 탭
  [1] 그룹 생성
  [2] 그룹 권한 조회 수정 삭제

5. 연동 관리 탭 (보완 필요)
  [1]외부 도구와의 연동 상태 관리

6. 보안 설정 탭 (보완 필요)
  [1]워크스페이스 보안 정책

7. 감사 로그 탭 (보완 필요)
  [1]워크스페이스에서 발생한 작업 내용들 추적
  [2] 최근 활동 내역 미리보기

8. AI 관리 탭
  [1] AI 어시스턴트
    (1) 학습 제어
    (2) 데이터 관리
    (3) AI 모델 설정
    (4) 학습 내역 로그 / 이력
    (5) 시스템 관리 / 보안
  [2] 검색 엔진 + AI
    (1) 채널 채팅 내역 학습 스케줄 설정
```

**확인 필요**:
